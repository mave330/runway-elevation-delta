Runway Elevation Delta

Saisis un code aéroport ICAO. L’application télécharge automatiquement les données pistes depuis OurAirports, projette un point dans l’axe extérieur de chaque QFU à la distance choisie, puis compare l’altitude terrain avec l’altitude du seuil.

Prêt. Exemple : LFPG, LFPO, LFML, LFLL, KLAX.
